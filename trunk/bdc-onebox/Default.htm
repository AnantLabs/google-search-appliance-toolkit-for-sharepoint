To build: use Visual Web Developer 2005 express edition

Create a website or virtual directory pointing to the root "bdc-onebox" Before you run: set the ssp - shared service provider - in web.config
Call: OneBoxHandler.ashx
Parameters:
lob= line of business name
entity= the "table"
fieldname= on which the search is performed
query= customer search terms

Example: http://contentvm1:8888/OneBox/OneBoxHandler.ashx?lob=AdventureWorksSampleInstance&entity=Product&fieldname=name&query=Bike
